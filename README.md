# Worker-Management-Application
This is a mini project for my Application Development subject at university.
A point that highlight out project is an option to add workers automatically when creating a task. It depends on task job to add workers who is working in that job.

My part in this project including both front-end(user interface) and back-end for assignment management(phân công lao động) and task management(quản lí công việc).
In addition, as leader of team, my responsibility was connecting team members and helping with their tasks too.
